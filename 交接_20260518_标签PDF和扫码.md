## 字体子集再生方法(将来需要扩字)

```bash
pip3 install --user fonttools brotli
# 1. 下载完整 NotoSansSC 变量字体
curl -L -o /tmp/NSC.ttf "https://github.com/google/fonts/raw/main/ofl/notosanssc/NotoSansSC%5Bwght%5D.ttf"
# 2. instance 到 Regular
~/Library/Python/3.9/bin/fonttools varLib.mutator /tmp/NSC.ttf wght=400 -o /tmp/NSC-Static.ttf
# 3. subset(把 /tmp/subset_chars.txt 改成需要的字)
~/Library/Python/3.9/bin/pyftsubset /tmp/NSC-Static.ttf \
  --output-file=fonts/NotoSansSC-Regular-subset.ttf \
  --text-file=/tmp/subset_chars.txt \
  --layout-features='' --no-hinting --desubroutinize --notdef-outline
```

当前覆盖:ASCII + Latin-1 + GB2312 (6763) + JIS X 0208 (~7000) + CJK 标点/全半角/通用标点 = ~10800 字,3.2MB。
