## PWA 使用方法

iPhone Safari:
1. 打开 `https://learnmans-inventory.vercel.app/inventory.html`
2. 底部分享 → **添加到主屏幕**
3. 主屏出现金色「**矿**」icon + 标签「**矿珍库**」
4. 点击启动 = 全屏 standalone 模式(无 Safari 工具栏)

资源在 `icons/`:
- `app-icon.svg`(源,矿字 + MINERAL VAULT 副标 + 金渐变)
- `apple-touch-icon.png` 180×180(iOS 必需)
- `icon-192.png` / `icon-512.png`(manifest maskable)
- `favicon-32.png`(浏览器 tab)

无 service worker(离线另算)。

## 扫码 v5 关键变化

**为什么改:**Quagga2 假阳性严重(把 CODE128 误读成短数字 `11924470` 之类)。改成:

- 路径 A: **BarcodeDetector**(Chrome Android / Edge 原生,准且快)
- 路径 B fallback: **jsQR + ZXing**(iOS Safari 没原生 BD)
- **ROI 裁剪**:从视频中心取 60% 短边正方形,匹配 `.scan-frame` 黄框,大幅降误识
- **requestVideoFrameCallback**:帧率跟着视频走(iOS Safari 15.4+),省 CPU
- **手电筒按钮** `#scan-torch-btn`:`track.applyConstraints({advanced:[{torch:true}]})`,仅 cap.torch=true 设备显示
- 解码 canvas 最大 720×720(平衡分辨率与速度)
